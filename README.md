# Unity-3D-RPG-Practice
# 🎮 3D RPG Tutorial (Unity 6)

블소 스타일 3D Action RPG “기본 시스템” 구축 튜토리얼/프로토타입.

## 📌 Overview
- Engine: Unity 6.0 (6000.0.48f1)
- Period: 2025.06.06 ~ 2025.07.25
- Goal: 블소 스타일 3D Action RPG 기본 시스템 구축

## 🎥 Demo
- YouTube: [(유튜브 링크)](https://youtu.be/MW7Uwm47NvU)

## ✨ Core Design
### Combat
- 전투 방식: 오토타겟팅 + 논타겟(실시간 액션)

### Controls (예시)
- 좌클 2연격: 기본공격(자원 충전)
- 우클: 강공격(자원 소모)
- 1~4: 역할별 스킬 슬롯

## ✅ Implemented / Progress
- CharacterController 기반 이동(WASD)
- 카메라 방향 기준 이동(Quaternion.AngleAxis 활용)
- Cinemachine FreeLook 세팅
- 입력 시스템(InputReader SO) 구성
- 기본 UI(체력/경험치 등) 연동
- 오토타겟팅(초기 버전) 구성
